Disable_DNSCryptProxy

About

Information Technology at Washington University in Saint Louis uses Cisco Umbrella for cloud security. This software makes use of DNSCrypt Proxy software to set up a local DNS proxy server which sometimes interferes with authentication into the Washington University in Saint Louis Databricks platform.

Solutions

DisallowRun_DNSCryptProxy.reg

The DisallowRun_DNSCryptProxy.reg file is a Windows registry file that disallows DNSCript Proxy from executing.

Task

The task_scheduler directory contains files for scheduling and unscheduling a Windows Task Scheduler task to run at system startup that renames the dnscryptproxy.exe and dnscrypt-proxy.exe executables. This prevents subsequent calls from finding and executing them. The script also stops any running DNSCrypt Proxy processes.
